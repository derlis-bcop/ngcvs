# CHECKLIST CETI — CHECKLIST DE ELEGIBILIDAD TÉCNICA INICIAL  
**Nivel 3 — Obligatorio para iniciar cualquier proyecto**  
Gerencia de TICs

El CETI debe completarse antes de iniciar actividades técnicas formales.  
El proyecto NO podrá avanzar si no obtiene estado **APROBADO** o **APROBADO CON OBSERVACIONES**.

---

# 1. Gobernanza y Cumplimiento
- [ ] Proyecto registrado formalmente  
- [ ] Roles definidos (Dev, QA, Arquitectura, Seguridad, Producción)  
- [ ] Se utilizará control de versiones SVN  
- [ ] Se utilizará CI/CD institucional  
- [ ] Se declara cumplimiento obligatorio de NGCVS, Sub-Políticas y Manuales  

---

# 2. Viabilidad Técnica Operativa
- [ ] Arquitectura preliminar definida  
- [ ] Modelo preliminar de despliegue identificado  
- [ ] Requerimientos de infraestructura estimados  
- [ ] Requerimientos de red (entradas/salidas) identificados  
- [ ] Dependencias internas/externas mapeadas  
- [ ] Evaluación preliminar de performance  

---

# 3. Observabilidad Inicial
- [ ] Estrategia preliminar de métricas técnicas definida  
- [ ] Estrategia preliminar de métricas de producto definida  
- [ ] Estrategia de logging estructurado definida  
- [ ] Estrategia de telemetría/tracing definida  
- [ ] Health checks planificados  

---

# 4. Seguridad
- [ ] Evaluación inicial de riesgos  
- [ ] Identificación de datos sensibles  
- [ ] Estrategia de Secrets Management definida  
- [ ] Validación preliminar con Seguridad TICs  
- [ ] Estrategia de autenticación/ autorización acordada  

---

# 5. Producción / Infraestructura
- [ ] Producción participó en la fase de idea  
- [ ] Validación preliminar de capacidad  
- [ ] Validación preliminar de requerimientos de despliegue  
- [ ] Validación preliminar de monitoreo  
- [ ] Estrategia preliminar de rollback aceptada  

---

# 6. Riesgos del Proyecto
- [ ] Riesgos técnicos documentados  
- [ ] Riesgos operativos documentados  
- [ ] Medidas de mitigación definidas  
- [ ] Restricciones operativas identificadas  

---

# 7. Dictamen Final

- [ ] **APROBADO**  
- [ ] **APROBADO CON OBSERVACIONES**  
- [ ] **NO APROBADO**
